# DeVoice 功能文档

## 目标

克隆 `https://devoice.io` 的产品体验，让当前项目从旧产品定位改为 DeVoice 风格的在线 AI 音视频工具站。

## 已覆盖范围

- DeVoice 公开站点外壳：侧边栏、促销条、签到、登录弹窗、积分和账号菜单。
- DeVoice 公开页面：首页、工具页、定价页、博客、隐私政策、退款政策和使用条款。
- 登录态页面：控制台、我的资源和任务结果页。
- API 路由：任务、上传、导出、计费、API Key 和健康检查。
- Worker 处理：媒体任务 Worker 和多服务商降级。

## 页面结构

工具页统一由 `components/devoice-tool-page.tsx` 配置驱动，`app/[locale]/[tool]/page.tsx` 同时承载工具页和静态内容页。

静态内容页由 `components/devoice-static-page.tsx` 提供：

- 博客
- 隐私政策
- 退款政策
- 使用条款

## 登录与账号

当前实现包含 DeVoice 风格登录弹窗，并提供基于 `DEVOICE_DEMO_EMAIL` 与 `DEVOICE_DEMO_PASSWORD` 的本地/演示 Credentials provider。生产级邮箱密码注册、重置密码和登录限流仍应新增安全的 password hash 模型后再开放。

## 积分

UI 复刻 DeVoice 的积分概念：

- 免费积分
- 付费积分
- 每日签到积分
- 一次性积分包

当前后端仍使用现有 quota/workspace 字段承载额度边界，运行时文案已改为 DeVoice 积分语境。
