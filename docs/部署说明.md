# DeVoice 部署说明

## 运行单元

- Web/API：Next.js App Router。
- Worker：`workers/media-worker.ts`，需要可长时间运行的进程环境。
- 数据库：Prisma 兼容的 MySQL/TiDB。
- Redis：BullMQ 队列和可选 REST 缓存。
- 存储：Cloudflare R2 或 S3 兼容对象存储。

## 基础命令

```bash
pnpm install
pnpm prisma:generate
pnpm prisma:push
pnpm build
pnpm start
```

Worker：

```bash
pnpm worker
```

## 注意

生产环境需要配置真实数据库、Redis、R2、AI provider 和 Stripe 密钥。未配置时，页面仍可构建和浏览，但上传、队列、计费和 AI 处理不可完整执行。
