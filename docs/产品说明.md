# DeVoice 产品说明

DeVoice 是一个在线 AI 音频、视频和 YouTube 工具产品。本项目复刻了 DeVoice 的核心产品页面、工具入口、积分定价、资源历史和结果页体验。

## 核心页面

- 首页：免费语音转文字。
- AI 转写：音频转文字、视频转文字、AI 语音转文字。
- AI 语音：背景降噪、文本转语音、AI 语音生成、AI 声音克隆。
- AI YouTube：YouTube 转写生成、YouTube 字幕下载、YouTube 视频摘要。
- 资源页：我的资源。
- 定价页：选择套餐，包含入门、标准、综合和精英积分包。
- 内容页：博客、隐私政策、退款政策和使用条款。

## 产品体验

- 左侧 DeVoice 导航栏。
- 每日签到积分弹窗。
- 登录弹窗和登录后账号菜单。
- 积分、购买、定价和我的资源入口。
- 上传、YouTube 链接、降噪、文本转语音和声音克隆的 DeVoice 风格操作面板。

## 后端能力

- NextAuth 登录会话。
- Prisma 数据模型。
- 媒体任务创建、状态查询和导出。
- Cloudflare R2 签名上传。
- BullMQ worker 处理入口。
- 多 AI 服务商降级。
- Stripe checkout/portal/webhook 路由。
- API Key 和开发者 API。
