# DeVoice 服务商接入与降级策略

DeVoice 站点保留多服务商降级机制，以保证转写、摘要和翻译任务在单个服务失败时仍有备选路径。

## 转写

优先级：

1. Deepgram
2. AssemblyAI
3. Groq Whisper

相关环境变量：

```bash
DEEPGRAM_API_KEY=""
ASSEMBLYAI_API_KEY=""
GROQ_API_KEY=""
```

## 摘要

优先级：

1. DeepSeek
2. Gemini

相关环境变量：

```bash
DEEPSEEK_API_KEY=""
GEMINI_API_KEY=""
```

## 翻译

优先级：

1. DeepL
2. DeepSeek 快速模型

相关环境变量：

```bash
DEEPL_API_KEY=""
DEEPSEEK_API_KEY=""
```

## 队列

后台任务使用 BullMQ。建议使用 Redis TCP 连接运行 Worker；页面层可用 REST Redis 做最近任务缓存。

```bash
REDIS_URL=""
DEVOICE_QUEUE_NAME="devoice-media-jobs"
```

## 存储

上传文件通过 Cloudflare R2 或 S3 兼容签名 URL 写入。

```bash
R2_ENDPOINT=""
R2_ACCESS_KEY_ID=""
R2_SECRET_ACCESS_KEY=""
R2_BUCKET=""
R2_PUBLIC_BASE_URL=""
```
